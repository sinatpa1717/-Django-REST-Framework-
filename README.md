پروژه فروشگاه آنلاین (Django REST Framework)

پیاده‌سازی سیستم ثبت‌نام کاربر (User Registration)

طراحی API به صورت RESTful برای اتصال به فرانت‌اند (وب یا موبایل)

مدیریت داده‌ها و تصاویر محصولات با Django و DRF
